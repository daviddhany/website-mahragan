# Feature Update Notes

This package was generalized into an English Student Activity Management System.

Included capabilities:

1. Student registration and generated student codes.
2. Student login and first-login password change.
3. Admin and teacher roles.
4. Coordinator role support.
5. Activity categories and activity management.
6. Team creation, lock/unlock, and student moving/removal.
7. Registration open/close control with optional scheduled close time.
8. Student file uploads for profile photo, birth certificate, and payment receipt.
9. Payment confirmation tracking.
10. Reports and export endpoints.
11. Clean URL routing.
12. English generic UI text suitable for resale.
